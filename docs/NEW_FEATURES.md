**Code of Omertà**: Privacy as a first-class contract, not a vibe.

### Core Principles:

1. **Collect only what is strictly needed** - Hard data minimization
2. **Never train on user data** - No global models
3. **Default to on-device inference** - No external API calls
4. **No "accept all"** - Everything locked down by default
5. **Full transparency and auditability** - Complete data flow visibility
